Задание

inheritance

Создать функцию-конструктор Person. В ней создать метод hello, который будет возвращать имя существа

Создать функцию-конструктор Boy и отнаследовать ее от Person. На вход она принимает строку (имя существа), которое должно возврашаться при вызове метода hello

var nick = new Boy('Nick'),
adam = new Boy('Adam');
adam.hello(); // Adam
nick.hello(); // Nick
